# Source Switch Commands

## Linux/macOS

```bash
# show current
bash skills/scripts/source/switch_source_cli.sh show

# pip mirrors
bash skills/scripts/source/switch_source_cli.sh pip tsinghua
bash skills/scripts/source/switch_source_cli.sh pip ustc
bash skills/scripts/source/switch_source_cli.sh pip default

# conda mirrors
bash skills/scripts/source/switch_source_cli.sh conda tsinghua
bash skills/scripts/source/switch_source_cli.sh conda ustc
bash skills/scripts/source/switch_source_cli.sh conda default
```

## Windows

```bat
cmd /c skills\scripts\source\switch_source_cli.bat show
cmd /c skills\scripts\source\switch_source_cli.bat pip tsinghua
cmd /c skills\scripts\source\switch_source_cli.bat conda default
```
