# Chemistry OB Commands

## Quick mixture calculation

```bash
python skills/scripts/chemistry/ob_mix_cli.py --mix "KClO4:65,Al:35"
```

## Direct full CLI

```bash
python skills/scripts/chemistry/ob_cli_v3.py --input "KClO4:65 Al:35"
python skills/scripts/chemistry/ob_cli_v3.py --optimize "Fe2O3 Al" --target 0
```

## Legacy interactive mode

```bash
python skills/scripts/chemistry/obcaculator.py
```
